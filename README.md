# Summary Report for the Arctic Ocean NRT and DM CTD Datasets

The site provides a collection of resources and summary reports containing statistics and other relevant information about the datasets.

## Website

The reports are hosted on [GitHub Pages](https://aiqc-hub.github.io/arc-report/).

## Data Sources

Three datasets were used to generate the reports:

### AR Dataset

- **Dataset ID**: [cmems_obs-ins_arc_phybgcwav_mynrt_na_irr](https://doi.org/10.48670/moi-00031)  
- **Duration**: History  
- **Region**: Arctic Ocean (AR)  
- **Category**: CT (CTD)

### GL Dataset

- **Dataset ID**: [cmems_obs-ins_arc_phybgcwav_mynrt_na_irr](https://doi.org/10.48670/moi-00031)  
- **Duration**: History  
- **Region**: Arctic Ocean (GL)  
- **Category**: CT (CTD)

### CORA Dataset

- **Dataset ID**: [cmems_obs-ins_glo_phy-temp-sal_my_cora_irr](https://data.marine.copernicus.eu/product/INSITU_GLO_PHY_TS_DISCRETE_MY_013_001)  
- **Dataset Name**: CORA  
- **Folder**: arctic
